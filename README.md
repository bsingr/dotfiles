dotfiles
========

## Features

Automated installation of configuration and tools. Supports following applications:
  
  * Terminal
    * Bash
    * ZSH
    * Git
    * Ruby (chruby) 
  * Sublime Text
    * Plugins
    * Themes 
  * XCode
    * Themes

## Installation

    git clone https://github.com/dpree/dotfiles.git ~/dotfiles
    cd ~/dotfiles
    rake
    
## Update

    cd ~/dotfiles
    git pull
    rake
